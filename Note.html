Coding is Funny!

strong : Bold word

u : underline

h1 ~ h6 : headline

img, input, br, hr, meta - 태그를 닫을 필요가 없음.

p : paragraph

HTML : 정보 표현

CSS : 정보 꾸밈

p style="margin-top:Npx;" - p 태그 위에 margin(여백)이 N(px) 만큼 생김.

img src="HYPER LINK" - src(source) 뒤에 하이퍼링크를 걸어주면 사진이 나옴.

웹페이지 파일(1.html)과 같은 위치에 있는 이미지 파일이라면 src 속성의 값으로 이미지 파일의 이름만 적으면 된다.

즉 img src="coding.jpg"라고 쓰면 된다.

하이퍼링크 뒤에 width="N%" 라고 쓰면 N에 100을 넣을 경우 100%의 크기로 출력됨.

li : list

ul : unordered list, 목록과 목록을 구분짓게 함. (점으로 표시)

ol : ordered list, 목록과 목록을 구분짓게 함. (숫자로 표시)

title : 웹페이지의 제목을 결정함.

meta charset="UTF-8" : UTF-8로 열어야 한다는 의미.

body : 본문

head : 본문 설명 태그

!doctype html : 웹페이지가 html로 만들어짐을 표현.

HTML(HyperText Markup Language)

a : anchor

href : HyperText Reference

target="_blank" : 링크를 클릭했을 때 새 창에서 페이지가 열리게 하는 속성

a href="LINK" : 해당 링크로 이동함.
